## Project includes:

- Vite.js (npm install, npm run dev)
- global context
- dark/light theme
- Unsplash API (https://unsplash.com/developers)
- React Query
- Local Storage
- ENV Variables

## Install

You can get it on npm:

1. npm install
2. npm run dev
3. and look at this website on localhost
